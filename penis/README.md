# Chrome AI Fact Checking Extension

A Chrome web extension in development and testing is designed to detect claims in user-visited articles, cross-check the statements with web-scraped data, and provide a legitimacy score.

## Installation

To experiment with this extension, please clone this repo and use 'Load Unpacked Extension'.
Read more on [Development Basics](https://developer.chrome.com/docs/extensions/mv3/getstarted/development-basics/#load-unpacked).
